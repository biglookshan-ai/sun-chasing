# SunPath v2.0 — 开发功能列表 · 计划 · 优化流程

面向**摄影 / 延时 / 影视 / 建筑 / 天文**的太阳 + 月亮方位规划工具。纯前端、无构建步骤、部署于 GitHub Pages。本文件既是路线图，也是本轮（v2.0）实际交付清单。

---

## 0. 竞品调研结论（决定我们做什么）

调研对象：PhotoPills、Sun Surveyor、The Photographer's Ephemeris (TPE / TPE 3D / TPE Web)、Shadowmap、Sun Seeker、PlanIt Pro。

**共识的核心价值（表 stakes）**
- 地图选点 + 太阳/月亮 rise-set 方位色线 + 方位角/高度角读数
- 时间轴拖动 + 高度曲线图（最被称赞的「高级感」交互）
- 黄金/蓝调时刻、暮光分级、月相
- AR 相机叠加太阳/月亮轨迹

**真正拉开差距的功能**
- **对齐引擎（PhotoPills Find）**：「太阳/月亮几点会落在这个位置」——最被称赞、也最难
- **地形遮挡**：「月亮是从山后升起还是山上方」——朴素 app 的通病
- **取景器 / 焦段模拟**：「我这支镜头里太阳会在画面哪个位置」
- **天气**：只有 PhotoPills 都没有的短板——黄金时刻能不能拍，看云量

**应用场景（use cases）**
1. 风光黄金/蓝调时刻——太阳/月亮落在特定山峰、拱门后
2. 房产/建筑——哪个立面几点被照亮、沿立面斜射出立体感、蓝调时段亮灯
3. 天文/银河——银心/银河拱桥对齐前景，配合新月
4. 人像——低角度太阳的逆光/轮廓光方向
5. 影视——多小时场景里太阳走向，规划布光
6. 无人机——鸟瞰视角的黄金时刻光向、日落合法飞行截止
7. 婚礼/活动——承诺客户魔法光线时段
8. 室内/光伏——窗户进光、屋顶/立面日照时数
9. 地标对齐——曼哈顿悬日、地标后满月

**高级 UI/UX 模式**
- 时间轴 = 太阳/月亮高度曲线图，拖动实时重绘
- 色彩语法：黄=日出 橙=日落 青=月出 蓝=月落，细线=当前位置
- 大字号太阳/月亮字形随光线阶段变色（金→紫→蓝→灰）
- 明确的 Day / Night 暗色主题

## 1. 完整功能清单（★=v2.0 本轮交付，☆=路线图）

- ★ 方位罗盘（太阳 + 月亮 + 全天轨迹）
- ★ 全天高度曲线（太阳 + 月亮）
- ★ 机位 URL 分享
- ☆ 对齐引擎（Find）、地形遮挡、DEM 高程、光污染/Bortle

- ★ **Open-Meteo 接入**（免 key、CORS、按浏览器 IP 无限流）
- ★ 当前/逐时 云量（总/低/中/高）、气温、降水概率、能见度、风
- ★ **黄金时刻可拍性评分**（按日出/日落时刻云量启发式）

### D. 取景器 / 焦段模拟（本轮新增 · 摄影杀手锏）

- ★ 横/竖构图切换
- ★ **画面内太阳 + 月亮位置**（矩形取景框 + 三分线）
- ★ **全天太阳/月亮轨迹划过画面**（gnomonic 直线投影）
- ★ 相机指向：一键对准太阳/月亮，或手动方位+俯仰
- ☆ 景深/超焦距、ND 曝光、延时、星轨 NPF 计算器

### E. AR 实景
- ★ 相机叠加太阳全天轨迹 + 当前位置（已有）
- ★ **月亮轨迹 + 当前位置叠加**（本轮新增）
- ★ 自适应抗抖、锁定方向、对准太阳校准、随手机转动的小地图、回到现在（已有）
- ☆ WebXR、夜间银河 AR

### F. UI/UX（本轮全面重做）
- ★ 赛博/现代暗调主题：近黑背景 + 霓虹青/金/紫强调 + 玻璃拟态 + 辉光

- ★ 字体层级（Space Grotesk 展示 + JetBrains Mono 数据）

## 3. 优化流程（贯穿始终）

**正确性**
- 天文层用「90−|纬度−赤纬|」等几何铁律 + 已知算例回归验证（正午高度、日出方位、南北半球）
- 方位角约定统一：SunCalc 自南顺时针 → 全应用统一转为自北顺时针（`azDeg`），避免取景器镜像
- 月亮加视差 + 折射修正；月出/月落用 2 小时抛物线扫描

**性能**
- 罗盘/曲线/取景器每帧重绘控制在轻量 SVG/Canvas；AR 用 rAF
- 天气请求防抖（选点/移动地图后延迟触发），结果缓存按坐标+日期
- 传感器数据自适应平滑（静止死区）降重绘噪声

**健壮性**
- 天气/搜索/地图瓦片外部依赖全部 try/catch + 优雅降级（沙盒/离线不崩）
- 无相机/无传感器时，取景器与数据面板照常可用
- App Proxy/外部 API 失败不阻塞核心渲染

**体验**
- 每次外部操作有反馈（toast/加载态/按压）
- 关键交互（回到现在、对准太阳）图标 + 文字，避免纯图标看不懂
- 移动优先，safe-area 适配，一屏可达

**验证与上线**
- 每轮改动在预览服务器跑：无 console error、主视图不回归、移动视口截图存证
- 部署后轮询线上文件哈希 == 本地，确认 CDN 生效
- 缓存：JS/CSS 更新后用 `?v=` 校验，提示用户绕缓存
